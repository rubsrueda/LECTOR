# LECTOR

Pequeña app para pegar un texto y leerlo en voz alta usando la API de síntesis de voz del navegador.

## Uso

1. Abre `index.html` en tu navegador o sirve el directorio con un servidor estático.
2. Pega o escribe el texto que quieras escuchar.
3. Pulsa **Leer en voz alta** para iniciar la locución.
4. Usa **Detener** para cancelar la lectura actual.

## Desarrollo local

```bash
cd /home/runner/work/LECTOR/LECTOR
npm start
```

Después, abre `http://localhost:4173`.
